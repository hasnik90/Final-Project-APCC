Pre-conception carrier screening tool for Whole Exome Sequencing data

Input text files have to be in tab-delimited(.txt) form with 38 columns as specified in README.The Input file is an edited VEP annotated tab-delimited output of the GATK VCF files containing Whole Exome sequencing data. More information about the VEP fields used to annotate the VCF file can be found from the link below. Visit Ensembl.org Variant Effect Predictor
